Destructuring in javascript

Its a way to extract values from objects,arrays and put them into individual variables




Object destructuring

const person = {
name: "abrar",
age: 50,
lastName: "hussain"
};
let {name, age, lastName}= person;
document.write(name);
document.write(age);
document.write(lastName);

output is:
abrar
50
hussain

Array destructuring

let salery = ["25000", "2000", "10000"];
let [high, low, average] = salery;
document.write(low);
document.write(high );
document.write(average);

output is:
2000
25000
10000

another example of array destructure by using ...rest

let number = [10, 20, 30, 40, 50, 60, 70];
let [a, b, ...c] = number;
document.write(a);
document.write(b);
document.write(c);

the output is:
10
20
30,40,50,60,70
